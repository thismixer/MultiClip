# MultiClip

**MultiClip** - утилита для синхронизации буферов обмена машин на ArchLinux и macOS.

## Установка

### Arch Linux
Используйте ваш любимый AUR-helper:
```bash
yay -S mclip-bin
```

### macOS
```bash
brew tap thismixer/homebrew-tap
brew install mclip
```

## Использование

```bash
mclip - запуск в фоновом режиме
mclip --install - установка с автозагрузкой
mclip --uninstall - удалить из автозагрузки
mclip -v - запуск с выводом логов в терминале
mclip -devices - вывод подключенных устройств
```
